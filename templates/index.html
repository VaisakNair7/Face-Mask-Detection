This Face Mask Detector model was build by training pre trained MobileNetV2 using Tensorflow. The model was trained on 3.4k images of faces with and without mask using 'imagenet' weights. MobilenetV2 is a lightweight Keras application with low parameter and depth count. Keras Applications are deep learning models that are made available alongside pre-trained weights. These models can be used for prediction, feature extraction, and fine-tuning. The fornt end was bulit using Flask web framework, Html, CSS and javascript.
